# Script Generation Prompt

Input:

- Story analysis JSON.
- Use `story_blocks` as the primary planning source.

Output:

- A 650-750 Chinese character narration script.
- Start with a 3-second hook.
- Add a reversal every 8-12 seconds.
- End with suspense.
